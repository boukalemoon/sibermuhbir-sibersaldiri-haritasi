🌍 Global Cyber Threat Map
Global Cyber Threat Map is an interactive dashboard that visualizes cyberattacks in real time, showing sources, targets, and attack vectors across the globe.
🚀 Key Features
- Active Threats: Displays the number of ongoing attacks
- Top Attack Sources: Countries where most attacks originate
- Top Targeted Countries: Countries most frequently targeted
- Attack Vectors: Types of attacks (SSH Brute Force, DDoS, SQL Injection, etc.)
- Live Threat Log: Detailed event log with IP, port, and attack type


🛠️ Technologies Used
- Frontend: JavaScript/React (or your chosen framework)
- Backend: Node.js / Python (if server-side logic is included)
- Visualization: D3.js / Chart.js / Map libraries (Leaflet, Mapbox, etc.)
- Data Source: Simulated feed or real-time API for cyber threat data

⚙️ Installation & Usage
# Clone the repository
git clone https://github.com/alenpepaxx1/global-cyber-threat-map.git

# Navigate into the folder
cd global-cyber-threat-map

# Install dependencies
npm install

# Start the application
npm start


Open http://localhost:3000 in your browser.


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

- Monitor active cyber threats in real time
